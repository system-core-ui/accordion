# Changelog

## [0.0.0] - Initial Release
- Scaffolded `@thanh-libs/accordion` library.
- Implemented `Accordion`, `AccordionSummary`, `AccordionDetails`, and `AccordionGroup` components.
- Added support for `exclusive` accordion mode natively via context Provider.
- Fully simulated ARIA compliance (expand, region, button relationships).
- Built stories and automated tests.
